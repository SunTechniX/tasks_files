# 📁 Проект: Работа с файлами в Python

Практические задания по работе с файлами, папками и обработке исключений в Python.

## 🎯 Цель

Закрепить навыки:
- Безопасной работы с файлами через `with`
- Использования `pathlib` для путей
- Обработки исключений `try...except`
- Работы с переменными окружения

## 📋 Условия заданий

### Задача 1: Безопасный журнал событий
**Файл:** `task_01.py`

Напишите скрипт, который запрашивает у пользователя заметку и записывает её в файл `journal.txt`.

**Требования:**
- [ ] Используйте контекстный менеджер `with`
- [ ] Режим открытия: добавление в конец (`"a"`)
- [ ] Укажите `encoding="utf-8"`
- [ ] Обработайте `PermissionError` и `OSError`
- [ ] Каждая запись с новой строки

---

### Задача 2: Генератор структуры проекта
**Файл:** `task_02.py`

Создайте скрипт, который автоматически разворачивает структуру папок для нового проекта.

**Требования:**
- [ ] Используйте `pathlib.Path`
- [ ] Создайте папку `project_root` с подпапками: `src`, `tests`, `assets`
- [ ] Используйте `mkdir()` с `parents=True` и `exist_ok=True`
- [ ] Выведите абсолютный путь к папке `src`

---

### Задача 3: Умный читатель конфигурации
**Файл:** `task_03.py`

Программа читает настройки из `config.txt` или создаёт его если не найден.

**Требования:**
- [ ] Используйте `Path.read_text()` / `Path.write_text()`
- [ ] Обработайте `FileNotFoundError`
- [ ] При создании запишите настройки по умолчанию
- [ ] Обработайте `PermissionError` и `UnicodeDecodeError`

---

### Задача 4: Анализатор больших логов
**Файл:** `task_04.py`

Анализирует лог-файл и подсчитывает количество записей разных уровней.

**Требования:**
- [ ] Читайте файл построчно (не весь сразу)
- [ ] Используйте `with open()`
- [ ] Посчитайте: ERROR, WARNING, INFO
- [ ] Обработайте отсутствие файла и ошибки кодировки

---
